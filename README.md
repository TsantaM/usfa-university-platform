# 🌐 Université Saint François d’Assise - Plateforme Web

Ce projet est un site internet réalisé avec **HTML**, **CSS**, **JavaScript** et **PHP** pour l’Université Saint François d’Assise.  
Il permet de présenter l’université au public, tout en offrant une plateforme numérique pour les **étudiants**, **enseignants** et **l’administration**.

---

## 🚀 Fonctionnalités

### Public
- Présentation de l’université
- Découverte des différentes filières (départements)
- Informations de contact

### Utilisateurs connectés
#### Étudiants
- Consulter leur **emploi du temps**
- Voir leurs **notes**
- Suivre leurs **absences**
- Accéder à des **documents partagés** (si implémenté)

#### Professeurs
- Ajouter ou modifier les **notes**
- Mettre à disposition des **documents**

#### Administration
- Gérer les **absences**
- Mettre à jour les **emplois du temps**

---

## 📁 Structure prévue du site

- `index.html` : Page d'accueil
- `login.php` : Connexion / inscription
- `dashboard/` : Tableau de bord selon le rôle
- `departements.php` : (optionnel) Détail des filières
- `assets/` : Images, icônes, fichiers CSS/JS
- `includes/` : Fichiers PHP réutilisables (header, footer, base de données, etc.)

---

## 🛠️ Stack utilisée

- **Front-end** : HTML5, CSS3, JavaScript
- **Back-end** : PHP (vanilla PHP)
- **Base de données** : MySQL / MariaDB (à venir)
- **Hébergement** : Serveur local (XAMPP, WAMP, etc.)

---

## 📅 Deadline & Organisation

Ce projet est réalisé dans un délai de **1 mois et 2 semaines** avec un temps de codage limité chaque jour.  
L’objectif est d’atteindre une version fonctionnelle avec les principales fonctionnalités.
